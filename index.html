HTML Introduction

HTML is the standard markup language used for creating web pages. It provides a structured way to present content on the web.

  • HTML stands for HyperText Markup Language.
  • HTML documents are composed of HTML elements, which define the structure and content of the page.
                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>My Web Page</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                        <p>This is my first web page.</p>
                    </body>
                    </html>
HTML Elements

HTML elements are the building blocks of an HTML document and define the structure and content of the page.

  • Elements are represented by tags, such as <p> for paragraphs, <h1> for headings, and <div> for divisions.
                
                    <p>This is a paragraph.</p>
                    <h1>Heading 1</h1>
                    <div>This is a division.</div>
HTML Attributes

HTML attributes provide additional information about HTML elements and are specified within the opening tag.

  • Attributes modify the behavior or appearance of elements, such as the "src" attribute for specifying the source of an image.
                
                    <img src="image.jpg" alt="Description of the image">
                    <a href="https://www.example.com">Click here</a>
HTML Headings

HTML provides six levels of headings, ranging from <h1> (highest level) to <h6> (lowest level).

  • Headings are used to structure the content hierarchy and provide visual emphasis. 
                
                    <h1>Main Heading</h1>
                    <h2>Subheading</h2>
HTML Text Formatting

HTML tags like <strong>, <em>, <u>, <s>, and <code> provide various text formatting options.

  • These tags can be used to make text bold, italicized, underlined, strikethrough, or represent code in the text.
                
                    <p>This is an example of <strong>bold</strong> text.</p>
                    <p>This is an example of <em>italicized</em> text.</p>
HTML Lists

HTML provides two types of lists: ordered lists (<ol>) and unordered lists (<ul>).

  • Lists are used to group related items together and provide a structured way of presenting information.
                
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ul>
HTML Images

Images can be displayed in HTML using the <img> tag.

  • The "src" attribute specifies the source file of the image, and the "alt" attribute provides alternative text for screen readers or if the image fails to load. 
                
                    <img src="image.jpg" alt="Description of the image">
HTML Links

Links are created using the <a> tag and are used to navigate to different web pages or sections within the same page.

  • The "href" attribute specifies the URL of the target page or location on the same page.
                
                    <a href="https://www.example.com">Click here</a>
HTML Forms

HTML forms are used to collect user input, such as text fields, checkboxes, radio buttons, and more.

  • Form elements are enclosed within the <form> tag, and user input is sent to the server for processing.
                
                    <form action="process.php" method="POST">
                        <label for="name">Name:</label>
                        <input type="text" id="name" name="name">
                        <input type="submit" value="Submit">
                    </form>
HTML Tables

HTML tables are used to display structured data in rows and columns.

  • Tables are created using the <table> tag, with the <tr> tag representing a table row, and the <td> tag representing a table cell.
                
                    <table>
                        <tr>
                            <th>Header 1</th>
                            <th>Header 2</th>
                        </tr>
                        <tr>
                            <td>Cell 1</td>
                            <td>Cell 2</td>
                        </tr>
                    </table>
